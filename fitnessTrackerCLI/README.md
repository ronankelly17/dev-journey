Fitness Tracker CLI App

* A simple command-line fitness tracker built with JavaScript using Node.js.

Features:

* Add workouts with name and calories burned
* View all logged workouts
* See statistics:
    * Total calories
    * Average calories
    * High-calorie workouts (>300)
* Input validation for calories
* Clean, user-friendly output

Technologies Used:

* JavaScript
* Node.js
* prompt-sync (for user input)

How to Run:

1. Clone the repository
2. Install dependencies:
    npm install
3. Run the app:
    node index.js
